# 🎧 Spotify Feature List (for MERN Project)

## 🟢 1. Core Features (MUST HAVE – MVP)

These are the minimum to make your app usable.

### 👤 User Authentication

* Sign up / Login (JWT-based auth)
* Google login (OAuth – optional)
* Logout
* Password reset

### 🎵 Music Playback

* Play / Pause songs
* Next / Previous track
* Seek bar (forward/backward)
* Volume control

### 📂 Music Library

* Display songs list
* Album view
* Artist view
* Song metadata (title, artist, duration)

### 🔍 Search

* Search songs
* Search artists
* Search albums

### 📃 Playlist Basics

* Create playlist
* Add/remove songs
* Delete playlist
* View playlist
